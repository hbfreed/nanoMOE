
# stack-edu

Educational code from [HuggingFaceTB/stack-edu](https://huggingface.co/datasets/HuggingFaceTB/stack-edu), downloaded from Software Heritage S3.

Requires `pip install boto3` before running.

After running `prepare.py`:

- val.bin has ~4M tokens
- Includes Python, JavaScript, Java, C++, and C samples
